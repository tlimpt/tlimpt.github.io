Tertúlia de Finanças Pessoais

Este é o percurso planeado: Lisboa → Setúbal → Grândola → Sines → Lagos → Faro → Évora → Tomar → Coimbra → Viseu → Chaves → Porto → Aveiro → Runa → Lisboa.
